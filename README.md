# web3-mq-react

A react ui components for wbe3mq

# Getting Started

In the project directory. first, install the dependencies, you can run:

```bash
yarn 
```
Second, package the project:

```bash
yarn build
```

Go to the `expamle` folder and  run the development server:

```
cd example
npm i
npm run dev
```
